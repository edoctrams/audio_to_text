# Audio to Text Converter

This mini-project converts `.mp3` or `.wav` audio files into readable text using **Google Speech Recognition API**.

## Features
- Upload and convert audio files
- Supports `.mp3` and `.wav`
- Transcribes audio to human-readable text

##  Requirements
- Python 3.x
- Libraries: `speech_recognition`, `pydub`
